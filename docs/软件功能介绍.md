# 软件功能介绍

本节按页面拆分，逐项说明“做什么、怎么用、边界规则”。

## 1. 进程管理

### 1.1 历史记录
- 展示所有已经达到记录阈值的窗口/标签统计数据（总可见时长、焦点时长、最后出现时间）。
- 支持按列排序：名称、类型、进程、分类、总可见时长、焦点时长、最后出现。
- 字符串按拼音字典序比较；时长默认降序；最后出现默认从近到远。
- 支持分类标记（学习/娱乐/社交/休息/其他），分类写入本地并在下次识别到同一 `classificationKey` 时继续沿用。
- 支持按“标签”收纳历史数据（有标签的会形成可展开组，显示标签汇总时长与最近出现时间）。

### 1.2 当前进程
- 只显示“当前仍在打开”的进程/窗口对象。
- 字段与历史记录一致，并额外支持直接分配“标签”。
- 关闭的窗口会自动从该页消失。

### 1.3 删除记录模式
- 点击右上角“删除记录”后进入勾选模式。
- 表头复选框为“全选/反选”逻辑（再次点击会对当前列表做反选）。
- 再点“确认删除”会删除选中的监控记录（包含 profile/session/stat/tag-assignment 关联项）。

### 1.4 标签管理
- 新增标签：输入名称后按回车或“确认”创建；名称为空则取消创建。
- 编辑标签：支持改名，回车确认。
- 删除标签：删除后会移除该标签的分配关系与标签统计。

### 1.5 系统事件
- 自动记录并展示：启动、挂起、恢复、锁屏、解锁、关机。
- 每条事件附带时间戳与颜色标记，供统计页时间线联动查看。

### 1.6 识别调试
- 展示当前焦点对象的识别结果（显示名、键值、类型、进程、域名、分类）。
- 展示焦点原始对象 JSON，便于排查识别问题。

### 1.7 窗口识别规则（关键）
- 桌面会被识别为独立对象（`Desktop`），默认分类为“休息”。
- 文件管理器窗口（`explorer.exe` 的普通文件夹窗口）会被忽略，不参与记录。
- 浏览器原生窗口标题（如“和另外 N 个页面...”）会被忽略，不再作为独立记录源。
- 浏览器标签依赖扩展桥接数据：
  - 默认按域名合并（如多个 `bilibili.com` 标签视为同一对象，时长覆盖计）。
  - 若 URL 命中白名单规则，则按完整 URL（规范化后）作为独立对象记录。
- 窗口记录阈值可配置（默认 60 秒）：短于阈值的临时窗口不会写入历史统计。

## 2. 专注（番茄钟 + 专注事项）

### 2.1 专注事项
- 可创建“专注事项”卡片，每项包含：
  - 标题
  - 默认时长（分钟）
  - 窗口组（从已识别窗口中多选）
- 事项卡片显示该事项窗口组“今日累计焦点时长”。
- 支持编辑、删除、加入专注队列。
- 选择窗口时列表按“最后出现时间”优先排序（最近活跃优先）。

### 2.2 番茄钟（基于队列，不含固定休息段）
- 计时单位来自“队列项时长”，不再使用全局专注/休息时长。
- 队列支持：
  - 从专注事项添加
  - 修改每项时长
  - 删除项
- 循环次数 `<= 0` 表示无限循环；否则按次数循环后进入下一项。
- 到点行为：系统通知 + 到点提示音（可在设置里选声音与音量倍率）。
- 提供开始、暂停、重置、跳过。

### 2.3 偏离检测
- 仅当当前队列项配置了窗口组时启用偏离检测；未配置则明确显示“不检测偏离”。
- 偏离模式支持单按钮切换：
  - 连续：回到目标窗口后偏离计时清零。
  - 累计：回到目标窗口后保留累计偏离时长。
- 偏离阈值（分钟）可配置；达到阈值触发一次提醒（通知 + 偏离提示音）。
- 计时使用毫秒级时间差累计，减少定时器漂移造成的偏差。

## 3. 时钟

### 3.1 秒表
- 精度：0.01 秒。
- 功能：开始、暂停、断点、重置、结束并保存。
- 每个断点记录：
  - 断点时刻
  - 相对上一条增量
  - 备注（可编辑）

### 3.2 秒表记录
- 列表展示每次秒表会话（可重命名、删除、查看详情）。
- 默认记录名：`开始时间 - 结束时间`。
- 删除需要二次确认。

### 3.3 秒表记录详情
- 可继续编辑记录名和每个断点备注。
- 右上角“复制”支持三种格式：
  - 复制纯文本
  - 复制 Markdown 格式
  - 复制表格格式（制表符，适合粘贴到 Excel）
- 备注为空时按空字符串输出，不会自动写“无”。

### 3.4 倒计时
- 可创建多个倒计时任务（名称、时长秒数）。
- 每条任务支持开始/暂停、改时长、应用修改、删除。
- 到点后行为由“设置 > 通用配置”决定：
  - 保留并划线（点击卡片可删除）
  - 完成即删除
- 到点会触发系统通知和“倒计时到点提示音”。

## 4. 数据统计

- 顶部可切换展示维度：`显示性质` / `显示窗口`。
- 可选统计日期（默认今天）。
- 页面展示当前焦点窗口（实时）。
- 图表包含：
  - 扇形图：选定日期分布
  - 柱状图：焦点时长排行（Top 10）
  - 时间线：最近 12 小时会话 + 开关机事件
  - 热力图：近 30 天，按分类查看每日时长深浅

## 5. 待办列表与归档

### 5.1 待办列表
- 新建待办卡片，支持搜索与类型筛选。
- 任务类型：
  - 一次性
  - 重复（每日/每周/每月/自定义模式）
- 自定义模式格式示例：`0,2,3,1,-1`（执行/跳过交替，末尾 `-1` 表示循环）。
- 定时提醒支持“年/月/日可空 + 时分秒必填”：
  - 年空：每年该月日提醒
  - 月空：该年每月该日提醒
  - 日空：该年该月每天提醒

### 5.2 待办详情
- 点击卡片进入详情页。
- 可修改标题、提醒开关与提醒时间。
- 可编辑“心得记录”，输入后约 800ms 自动保存（防止丢失）。

### 5.3 完成与归档
- 勾选待办即完成，生成归档记录（包含心得快照和任务 JSON 快照）。
- 一次性任务：标记归档后不再出现在待办列表。
- 重复任务：完成后心得会清空，进入下一轮时为新心得。

### 5.4 归档列表与详情
- 按任务分组显示归档次数和最近完成时间。
- 进入详情可查看每次完成记录和快照。
- 删除归档组需要二次确认，且会清理已归档的一次性任务残留。

## 6. 设置

### 6.1 通用配置
- 记录阈值（秒）：窗口累计可见时长达到阈值才写入统计。
- 主题切换：暗色 / 亮色。
- 开机自启动：桌面版可用（写入系统登录项）。
- 倒计时完成后处理方式：保留划线 / 完成即删。
- 数据文件路径：
  - 显示当前数据文件绝对路径
  - 支持手动输入或文件选择器
  - 目标不存在时可弹窗确认“创建新数据文件并切换”
- URL 白名单（支持通配符）：命中后按 URL 级别识别浏览器标签。
- 进程黑名单（支持通配符）：
  - 可按名称/类型/进程任意组合匹配
  - 同一规则中填写的字段需同时匹配才忽略
- 清除所有数据：二次确认后清空本地全部业务数据。

### 6.2 提示音管理
- 可配置三类事件声音：
  - 番茄钟到点
  - 偏离提醒
  - 倒计时到点
- 每类可选择“无”（不播放）或选择已有声音文件。
- 每类可设置事件音量倍率（无上限）。
- 声音库管理：
  - 新增声音（系统文件选择器 / 手动路径）
  - 改名、改路径、改默认倍率
  - 点击行或“试听”按钮试听
  - 删除声音（若被事件引用会自动回退到其他可用声音）
- 内置两条默认声音文件（到点、偏离）会自动存在于声音库。

## 7. 浏览器扩展（Browser Bridge）

- 目录：`browser-extension/`
- 版本：`0.2.0`
- 用途：将浏览器标签信息上报到本机 `http://127.0.0.1:17321/browser-bridge`，用于标签识别。
- 规则：
  - 默认按域名识别（忽略路径、参数、锚点）
  - 白名单命中时按 URL 粒度识别

# 窗口专注助手（WPF）

## 功能概览
- 分页式界面：每页聚焦一个功能，可通过顶部按钮切换。
- 番茄钟：
  - 手动设置专注/休息时长。
  - 可设置专注开始时自动跳转到指定页面。
  - 可从当前打开窗口中选择“允许专注窗口”。
  - 偏离提醒支持阈值分钟数、连续/累计两种统计方式。
  - 支持专注事项库、专注队列、循环次数与模板保存/加载/删除。
- 窗口监测：
  - 记录所有可见窗口的开启时长。
  - 记录焦点窗口时长排行。
  - 支持桌面识别与浏览器标签页（前台）识别。
- 开关机/会话事件：
  - 记录系统启动、关机、挂起恢复、锁屏解锁等事件。
- 数据统计页面：
  - 进程性质映射（学习/娱乐/社交/休息/其他），可持久化保存。
  - 扇形图、时间表、热力图、柱状图。
  - 支持按“窗口性质”或“具体窗口”切换统计。
- 待办事项：
  - 卡片创建、编辑、归档、删除归档（二次确认）。
  - 周期规则（每日/每周/每月/自定义）与定时提醒。
  - 心得文本自动保存，防止丢失。

## 运行
```powershell
dotnet run
```

## 发布（单文件 EXE）
```powershell
.\publish.ps1
```

发布目录默认是：
`bin\Release\net8.0-windows\win-x64\publish`

如果目标 EXE 被占用，脚本会自动回退到带时间戳的新目录。

## 数据文件
程序会在可执行文件同级目录创建 SQLite 数据库：
- `data\app.db`
